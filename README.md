### 0. 로그인 화면
- UProvider / User 선택
- User 선택시 비회원 로그인 옵션 추가
- 이메일 / 비밀번호 회원가입/로그인 & Flutter 구글 회원가입
- 이메일 정보 없을 시 "등록된 이메일이 없습니다." 출력
- 비밀번호 입력 오류시 "비밀번호가 잘못되었습니다. " 출력

## <USER VERSION>
### 1. 배경(메인 화면)
- 최상단 : 지역명 | 남은 이용권 수 | 로그인(마이페이지)
- 상) 카테고리
- 중) (1)인기업체 3곳 이미지슬라이더 
	(2)설명
- (1), (2) 병렬배치
- 하) 공지사항

### 2. 카테고리
- 대분류 [스포츠 / 미술&공예 / 음악 / 기타]
- 소분류 [야구. 농구. 탁구 / 드로잉.목공. 캐리커쳐 / 건반. 드럼. 보컬 / 기타] 
- 각 소분류 카테고리 옆에 별or하트 아이콘 클릭 시 
(1) 아이콘 채색 
(2) 관심 카테고리에 등록 
- 각 카테고리 선택시 해당 업체들 정보 나열

### 3. 마이페이지 _ 이용권 충전
- 충전
- 결제는 구현 x -> 결제 버튼 누르면 이용권 지급

### 4. 이용권 차감
(1) QR코드 촬영 버튼 -> QR 촬영시 이용권 차감 & Provider에 알림

## <PROVIDER VERSION>
### 1. 배경 (메인 화면)
- 출석부(?) | 매장관리  버튼 좌우 병렬 배치

### 2. 출석부
- USER이 QR코드를 스캔하면 이용자 정보를 출력

### 3. 매장관리
- 사진을 등록할 수 있음
- 매장 정보를 입력할 수 있음
